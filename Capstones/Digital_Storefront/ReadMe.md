# Collab Computers Website

## Project Overview
The Collab Computers website is a multi-page business website designed to showcase computer products and services. The website provides information about custom computers, repair services, upgrades, and customer reviews.

The goal of this project is to demonstrate front-end web development skills using HTML, CSS, and Bootstrap while creating a professional and responsive business-style website.

This project was created as part of a web development assignment to practice building structured, responsive webpages.

## Features

### Home Page
- Introduction to the Collab Computers brand
- Main navigation to the rest of the website
- Responsive layout

### Services Page
Displays available computer services using Bootstrap cards.

Services include:
- Custom Computer Builds
- Computer Repair
- Hardware Upgrades

Additional service pricing examples:
- Diagnostic Checks
- Virus Removal
- Operating System Reinstall
- RAM Upgrades
- SSD Installation
- Graphics Card Installation

The services page also includes **Quick View modals** to provide more detailed descriptions of each service.

---

### About Page
Provides background information about the company including:
- Business purpose
- Founding story
- Mission of the company

---

### Reviews Page
Displays customer testimonials using Bootstrap cards.

Features include:
- Star ratings using Bootstrap Icons
- Multiple review entries to simulate customer feedback
- Responsive card layout

---

### Contact Page
Allows users to contact the business.

Includes:
- Contact information
- Email link
- Phone number
- Social media links

---

### Footer
Each page includes a footer containing:
- Company description
- Contact information
- Social media links
- Privacy policy link

## Technologies Used

The following technologies were used to build this project:

- **HTML5** – Page structure and content  
- **CSS3** – Custom styling  
- **Bootstrap 5.3** – Responsive layout and components  
- **Bootstrap Icons** – Star ratings and social media icons  

External resources used:
- Bootstrap CDN  
- Bootstrap Icons CDN  
- Image assets used for branding and layout  

---

## Responsive Design

Bootstrap's grid system was used to ensure the website works across different screen sizes including:

- Desktop  
- Tablet  
- Mobile devices  

Components such as the navigation bar, cards, and layout grids automatically adjust depending on the screen width.

## How to Run the Project

1. Download or clone the project files.  
2. Open the project folder.  
3. Double-click **index.html** or open it in a web browser.

Because the website uses static HTML files, **no server setup is required**.
